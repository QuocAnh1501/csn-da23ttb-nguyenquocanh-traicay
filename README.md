# Website Bán trái cây nhập khẩu

## Giới thiệu
Đây là website quản lý và giới thiệu các loại trái cây nhập khẩu và trong nước.  
Website cho phép hiển thị hình ảnh, giá bán, xuất xứ, mô tả sản phẩm và liên kết QR truy xuất nguồn gốc.

## Công nghệ sử dụng
- HTML, CSS, JavaScript
- PHP
- MySQL
- phpMyAdmin
- XAMPP

## Chức năng chính
- Hiển thị danh sách sản phẩm trái cây
- Thêm / sửa / xóa sản phẩm
- Hiển thị hình ảnh trái cây
- Lưu thông tin: tên, giá, xuất xứ, mô tả
- Gắn link QR truy xuất nguồn gốc
- Thanh toán sản phẩm
## Cơ sở dữ liệu
- Database: "traicay_db"

## Hướng dẫn chạy project
1. Cài đặt XAMPP
2. Chạy Apache và MySQL
3. Import file `traicay_db.sql` vào phpMyAdmin
4. Copy thư mục project "traicay" vào "htdocs"
